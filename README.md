# CSC4303_Final_Project
Final Project for Dr. Uju's CSC4303 - Database Management Systems class.

__Our group has selected to build a "Movie Streaming Platform" database__

*The system manages users, movies, subscriptions, and viewing activity. Users access content through a streaming
application, while all media data, recommendations, and analytics are handled by a backend server connected to a
relational database.*

**User Roles (Client Side):**
  User / Subscriber
    • Browse and search movies
    • Watch content and manage watchlists
    • Rate and review movies
    • View recommendations
    • Manage subscription plans
  Content Manager
    • Add and manage movies and metadata
    • Update content availability
    • Categorize content by genre
  Data Analyst
    • Analyze viewing behavior and trends
    • Generate recommendation insights
    • Monitor platform engagement
  Administrator / Platform Manager
    • Manage users, subscriptions, and roles
    • Monitor system performance and security
    • Generate reports on usage and revenue
    • Set subscription and content policies
